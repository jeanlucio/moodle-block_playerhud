# 🧩 Optional Integration: Late Penalty

The **Deadline Extension** item power (see [Features](features.html)) does not work on its own — it requires the separate **Late Penalty** plugin (`local_latepenalty`, by the same author, but **not** part of the PlayerGames family). When installed, redeeming a Deadline Extension item pushes back the activity's effective deadline for that student and triggers Late Penalty's automatic recalculation, waiving or reducing any late-submission grade penalty already applied. Without Late Penalty installed, the item power fails gracefully with a "not installed" message instead of granting the extension.

👉 https://github.com/jeanlucio/moodle-local_latepenalty
